Describe your color scheme

The color scheme is based on the ocean, because I went with a theme of traveling to far off places. It uses a lot of blue, which goes to a seaweed greenish color in a few places to accent it. The only other colors used are a whitish green, just so I had fairly light color to use, and didn't clash too badly with anything. I also just put all the colors used in root as variables so I could change them at any point. (color near each other on the wheel). As for other formatting, in howto.html I did float an image over to the left so text went around it, used a couple pseudo classes, mainly :hover, but also :before and :target.

Typography / proximity

(required explanation) Spacing was important in the How To page to differentiate different articles on the same page, and each is separated by a good margin-bottom for each and a bold title. I also made sure to add some margins between text and images, so that both were distinct and readable. Also, in order to get the navigation compass to work correctly, I had to use a lot of strict positioning and containers, and the compass icon itself is a background, for a div, using contain to fit the size of the circle (made using border-radius) it's in and positioned halfway up and over from its original positioning. In order to make the rotation work, used some js and the before pseduo-selector.
